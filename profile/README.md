## Troubleshooting Common Issues

Install issues? Run the installer as administrator and temporarily disable any third‑party shell extension managers that may block context‑menu registration.  
Performance problems? Enable multi‑threading in Settings > Compression and switch from RAR4 to RAR5 for better dictionary utilisation on multi‑core CPUs.  
License/activation? A WinRAR license is per‑user, not per‑machine, and the key file goes into the program directory or the user's roaming profile.  
Compatibility? RAR5 archives require WinRAR version 5.0 or later; recipients on older versions will see an error and need an upgrade.  
Update failures? Download the latest installer manually and run it over the existing installation — settings and license files are preserved.
